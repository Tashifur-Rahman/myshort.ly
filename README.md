# 🔗 myshort.ly

**myshort.ly** is a full-stack URL shortener that transforms long URLs into short, shareable links within seconds.
Designed with a custom retro-inspired UI, this project combines functionality with an engaging user experience, making it a practical demonstration of full-stack web development.

Built using **HTML**, **CSS**, **JavaScript**, **Node.js**, **Express.js**, **MySQL**, and **NanoID**, this project helped me understand how the frontend, backend, and database work together in a real-world web application.

---

## ✨ Features

* 🔗 Generate short URLs instantly
* 🚀 Unique short codes using NanoID
* 💾 MySQL database integration
* 🔄 RESTful API built with Express.js
